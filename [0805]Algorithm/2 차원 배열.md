# 2 차원 배열

2차월 배열의 선언

> 1차원 list를 묶어놓은 list
>
> 2차원 이상의 다차원 list는 차원에 따라 index를 선언
>
> 2차원 list의 선언 : 세로길이 (행의 개수), 가로길이(열의 개수)를 필요로함
>
> - 행을 우선 입력. 그다음에 열
> - 행세로 열가로

```python
# 1차원 배열 초기화
arr = [0] * 5
#1차원 배열 받기 => 
for i in range(5):
	arr[i] = list(map(int, input().split()))
    
print(arr[i]) # -> 1차원 출력
print(arr) #-> 2차원 출력
print(arr[0][1]) # 2차원 배열의 인덱스에 접근

#각각 데이터 다 출력
for i in range(5):
    for j in range(len(arr[i])):
        print(j)
```
